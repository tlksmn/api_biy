## Dependencies
To install dependencies run in console
```bash
yarn install
```
or
```bash
npm install
```

## Environments
To start project be sure for start postgres database in port 5432 and edit account user data in .env files
```dotenv
#open for development mode file in env/.env.dev and write data
POSTGRES_USERNAME=postgres
POSTGRES_PASSWORD=example
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DATABASE=postgres
JWT_EXPIRATION_TIME=86400 #seconds
JWT_SECRET=jwtSecret
RABBIT_PASSWORD=rabbitPasswordForEncryptSellerPassword
ADMIN_PASSWORD=passwordForAdminPanel
```
